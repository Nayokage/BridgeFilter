# Bridge Filter

Minecraft мод для фильтрации и форматирования сообщений от Bridge ботов.

## Описание

Bridge Filter - это клиентский мод для Minecraft, который позволяет:
- Фильтровать сообщения чата по списку заблокированных ников/слов
- Подсвечивать ваш ник в сообщениях чата
- Автоматически форматировать сообщения от Bridge ботов (Minecraft, Telegram, Discord)

## Версии

- **Minecraft**: 1.21.1
- **Fabric Loader**: 0.16.9+
- **Java**: 21

## Установка

1. Установите Fabric Loader для Minecraft 1.21.1
2. Скачайте Fabric API и Cloth Config API
3. Поместите `bridgefilter.jar` в папку `mods`
4. Запустите игру

## Использование

- **Открыть меню настроек**: Правый Shift
- **Конфигурация**: Файл `config/BridgeFilter.json` создается автоматически

## Функции

### Фильтрация сообщений
- Блокировка сообщений по списку ников/слов
- Работает только с сообщениями от Bridge ботов
- Проверка по тексту сообщения и нику отправителя

### Подсветка ника
- Автоматическая подсветка вашего ника в сообщениях
- Настраиваемый цвет подсветки
- Поддержка замены ника через другие моды

### Форматирование Guild Bridge
- Автоматическое форматирование сообщений от Bridge ботов
- Определение источника (Minecraft, Telegram, Discord)
- Цветовое кодирование по источнику
